# Suporte

## Como pedir ajuda

- Bugs: abra uma issue com passos de reproducao, comportamento esperado/atual e logs relevantes.
- Perguntas: use Discussions (se habilitado) ou uma issue com o prefixo `[question]`.

## Politica

- Este repositorio e um **demo app**. Nao ha SLA.
- Vulnerabilidades: siga `SECURITY.md` (nao abra issue publica).
