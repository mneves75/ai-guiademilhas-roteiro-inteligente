# Codigo de Conduta

Este projeto adota o **Contributor Covenant**, versao 2.1.

Texto completo (PT-BR): https://www.contributor-covenant.org/pt-br/version/2/1/code_of_conduct/

## Aplicacao

Para relatar violacoes:

- Use os canais descritos em `SECURITY.md` quando o relato envolver risco de seguranca.
- Caso contrario, abra uma issue privada (se disponivel) ou contate os mantenedores pelos canais descritos no `README.md`.

## Atribuicao

Este Codigo de Conduta e adaptado do Contributor Covenant, versao 2.1.
