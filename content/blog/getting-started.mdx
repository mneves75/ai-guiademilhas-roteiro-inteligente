Welcome to NextJS Bootstrapped Shipped! This guide will help you get up and running with your new project in minutes.

## Prerequisites

Before you begin, make sure you have:

- Node.js 20+ and pnpm
- A PostgreSQL database (we recommend [Neon](https://neon.tech))
- A Stripe account for payments

## Quick Start

1. Clone the repository and install dependencies:

```bash
git clone https://github.com/your-org/nextjs-bootstrapped-shipped.git
cd nextjs-bootstrapped-shipped
pnpm install
```

2. Copy the environment file and configure your variables:

```bash
cp .env.example .env.local
```

3. Set up your database:

```bash
pnpm db:push
pnpm db:seed
```

4. Start the development server:

```bash
pnpm dev
```

## Project Structure

The project follows a clean, organized structure:

- `app/` - Next.js 16 App Router pages and layouts
- `src/components/` - Reusable UI components
- `src/lib/` - Utility functions and configurations
- `src/db/` - Database schema and queries
- `content/` - MDX blog content

## Key Features

### Authentication

We use Better Auth for authentication, supporting:

- Email/password login
- OAuth providers (Google, GitHub)
- Magic links
- Session management

### Multi-Tenancy

Built-in workspace support with:

- Team invitations
- Role-based access control
- Workspace isolation

### Payments

Stripe integration includes:

- Subscription management
- Customer portal
- Webhook handling
- Usage-based billing support
